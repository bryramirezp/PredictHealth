🏗️ Estructura Frontend

La estructura del frontend está organizada de manera clara y lógica. Todo está dividido en carpetas y archivos que hacen fácil encontrar y modificar el código.

¿Cómo está organizado el frontend?

Imagina que el frontend es como una casa. Tiene diferentes habitaciones (carpetas) donde cada cosa tiene su lugar. Los archivos HTML son las habitaciones, los CSS son la decoración, y los JavaScript son los electrodomésticos que hacen que todo funcione.

Carpetas principales

Templates

Los archivos HTML que contienen la estructura de las páginas web.

Static/CSS

Los archivos de estilos que hacen que las páginas se vean bonitas.

Static/JS

Los archivos de JavaScript que hacen que las páginas sean interactivas.

Static/Images

Las imágenes y fotos que se usan en las páginas.

Tecnologías utilizadas

El frontend utiliza tecnologías modernas y estandarizadas para crear una experiencia de usuario excelente. Solo se utilizan HTML5, CSS3 y JavaScript.

HTML5

HTML5 es el lenguaje que se usa para crear la estructura de las páginas web. Es la versión más moderna y tiene muchas características nuevas.

CSS3

CSS3 es el lenguaje que se usa para dar estilo y colores a las páginas web. Permite crear diseños modernos con animaciones y efectos visuales.

JavaScript

JavaScript es el lenguaje que hace que las páginas sean interactivas. Permite crear botones que responden, formularios que se validan, y mucho más.

Esta estructura ha sido diseñada para ser simple, eficiente y fácil de mantener a largo plazo.
