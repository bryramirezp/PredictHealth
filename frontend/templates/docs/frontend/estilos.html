🎨 Estilos y Diseño

El sistema de diseño de PredictHealth utiliza tecnologías modernas de CSS para crear una experiencia visual coherente y atractiva en toda la aplicación.

¿Qué es el sistema de diseño?

Imagina que el sistema de diseño es como las reglas de un juego. Todas las piezas (botones, tarjetas, colores) siguen las mismas reglas para que todo se vea bien junto. Esto hace que la aplicación sea fácil de usar y bonita de ver.

Tecnologías utilizadas

CSS3

La versión más moderna de CSS con variables, gradientes y animaciones.

Variables CSS

Colores y valores guardados en variables para cambiar todo fácilmente.

Efectos Visuales

Animaciones, sombras y efectos que hacen la interfaz más viva.

Responsive

La aplicación se ve bien en teléfonos, tablets y computadoras.

Variables CSS

Las variables CSS son como cajitas donde guardamos colores y valores. Si queremos cambiar un color, solo cambiamos la variable y todo se actualiza automáticamente.

¿Cómo funcionan las variables?

Las variables se definen al principio del archivo CSS con dos guiones (--). Luego se usan con la función var(). Por ejemplo: 

:root {
    --color-principal: #667eea;
    --espacio-grande: 2rem;
}

.boton {
    background: var(--color-principal);
    padding: var(--espacio-grande);
}

Colores y gradientes

Usamos una paleta de colores consistente con gradientes modernos. Los gradientes son mezclas de colores que van de uno a otro, creando efectos visuales atractivos.

Paleta de colores

  • Azul principal: Para elementos importantes como botones principales
  • Verde éxito: Para confirmar acciones positivas
  • Naranja advertencia: Para llamar la atención
  • Rojo peligro: Para errores o acciones peligrosas

Efectos visuales

Los efectos visuales hacen que la aplicación se sienta viva y moderna. Incluyen animaciones suaves, sombras y efectos de vidrio.

Animaciones

Las animaciones son movimientos suaves que responden a las acciones del usuario. Por ejemplo, cuando pasas el mouse sobre un botón, este se levanta un poco.

Sombras

Las sombras dan profundidad a los elementos, haciendo que parezcan flotar sobre la página. Usamos diferentes tipos de sombras para diferentes situaciones.

Tipografía

La tipografía es la forma en que se ve el texto. Usamos la fuente Inter porque es moderna, fácil de leer y se ve bien en pantallas.

Jerarquía del texto

  • Títulos grandes: Para los encabezados principales
  • Títulos medianos: Para secciones importantes
  • Texto normal: Para el contenido principal
  • Texto pequeño: Para información adicional

Diseño responsive

El diseño responsive significa que la aplicación se adapta al tamaño de la pantalla. En el teléfono se ve diferente que en la computadora, pero siempre se ve bien.

Puntos de quiebre

  • Móviles: Pantallas pequeñas, menú plegable
  • Tablets: Pantallas medianas, layout de dos columnas
  • Computadoras: Pantallas grandes, layout completo

¿Por qué es importante el sistema de diseño?

  • Consistencia: Todo se ve igual en todas partes
  • Velocidad: Los desarrolladores pueden trabajar más rápido
  • Calidad: La aplicación se ve profesional
  • Mantenimiento: Es fácil cambiar y mejorar el diseño

El sistema de diseño está en constante evolución para ofrecer la mejor experiencia posible a los usuarios.
